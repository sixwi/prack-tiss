<h2>Demonstration: Storing user input into a global variable</h2><br>
<input type="text" id="userName1" /> 
<input type="button" onclick="myFunction1()" value="Name?">
<p id="demo1"></p>

<!-- 
We create a variable, “theirInput”, that will hold the user’s input.
We add an event listener to “inputElement” that triggers on “change” 
and will fire an anonymous function when “change” is emitted.
The anonymous function sets “theirInput” to be equal to the event’s 
target’s value - which just so happens to be the text that was input by a 
user in the text field when they are no longer focused on the element 
~ Notes by Sean Grogg
-->

<script> 
  var inputElement = document.getElementById('userName1'); 
  var theirInput1 = ''; 
  inputElement.addEventListener('change', function(e) { 
  theirInput1 = e.target.value; 
  });  
  function myFunction1() {
  document.getElementById('demo1').innerHTML = 'Welcome, ' + theirInput1 + "!"
  }
</script>

<!-- 
This code makes a lot of assumptions and is not made for production use 
but rather solely as an example. Realistically, you’d likely want to 
operate on the value of the text field within the anonymous callback 
function rather than storing it globally. 
~Notes by Sean Grogg
-->
